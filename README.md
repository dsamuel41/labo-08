# labo-08
¿Cuál es la diferencia entre autenticación y autorizacion?
La autenticación verifica la identidad del usuario, y la autorización determina los permisos que tiene dentro del sistema.

¿Cuál es la función del token JWT en la guía?
La función del token JWT es verificar y mantener la identidad del usuario de forma 
segura después de iniciar sesión, permitiendo que acceda a recursos o rutas protegidas sin tener que autenticarse cada vez
